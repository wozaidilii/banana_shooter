# 香蕉突击

一个移动端 H5 第一人称 3D 小游戏。打开页面后会直接进入一局 45 秒的香蕉发射挑战，玩家用一根手指按住拖动瞄准，松开后把香蕉扔向猴子。

## 运行

Three.js 以本地模块形式放在 `vendor/three.module.min.js`。本地预览请用静态服务器：

```bash
python3 -m http.server 5174 --bind 127.0.0.1
```

然后打开 `http://127.0.0.1:5174/`。部署到 GitHub Pages 或任意 HTTP 静态托管后，手机浏览器点开即可玩。

## 玩法

- 按住屏幕拖动调整第一人称视角，松开手指发射香蕉。
- 命中远处或连续命中会得到更高分。
- 猴子靠近并没有拿到香蕉会扣血，同时打断连击。
- 45 秒后结算本局分数，并在本机记录最高分。
